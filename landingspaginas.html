Provincie Landingspagina's

Per provincie: de WordPress-titel, permalink, Yoast meta description en de HTML-inhoud om te kopiëren. Het WordPress-thema zet automatisch de header met het Employ-logo eromheen — je plakt alleen de inhoud.

Hoe aanmaken per provincie:
  1. WordPress → Pagina's → Nieuwe pagina
  2. Titel invullen (zie "WordPress-titel" hieronder)
  3. Permalink instellen op de slug (zie "Permalink")
  4. Schakel naar HTML bewerken (code-weergave, rechtsboven in editor)
  5. Klik op Kopieer HTML hieronder → plak in het codeveld
  6. Scroll naar Yoast SEO → plak de meta description
  7. Klik op Publiceren
